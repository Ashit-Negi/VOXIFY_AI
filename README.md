# Voxify AI

AI-powered Speech-to-Text SaaS application built with Next.js, Express.js, MongoDB, and Groq Whisper API.

## Features

- Audio Upload
- Live Audio Recording
- Speech-to-Text Conversion
- Transcript History
- Modern Dashboard UI

## Tech Stack

### Frontend

- Next.js
- Tailwind CSS

### Backend

- Express.js
- MongoDB
- Groq API
